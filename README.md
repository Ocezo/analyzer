# Image Consistency & Authenticity Analyzer

## Overview

A lightweight C++/OpenCV tool to analyze image consistency, detect edits, and estimate AI generation likelihood.

The goal is to determine:

1. Whether the two images belong to the **same real-world context**.
2. Whether one image is likely a **modified (cleaned) version** of the other.
3. The **probability that the images were generated by AI** (suspicion score).

**Important**

This system provides **probabilistic analysis**, not absolute proof.  
Results should be interpreted as **confidence scores**, not definitive conclusions.

---

## Features

### 1. Context Similarity Analysis
- Detects whether both images depict the same scene.
- Uses:
  - Feature matching (SIFT / ORB)
  - Geometric verification (homography / RANSAC)
  - Global color and texture similarity
- Outputs:
  - A similarity score ∈ [0, 1]

---

### 2. Image Derivation Detection (Object Removal / Editing)
- Determines whether the second image is derived from the first.
- Detects localized differences (e.g., removed objects like trash).
- Pipeline:
  - Image alignment
  - Pixel-wise and structural comparison
  - Difference localization analysis
- Outputs:
  - A derivation confidence score ∈ [0, 1]

---

### 3. AI Generation Suspicion Score
- Estimates the likelihood that an image was generated or heavily modified by AI.
- Uses heuristic indicators:
  - Noise consistency analysis
  - JPEG compression artifacts
  - Frequency domain anomalies
  - Metadata (EXIF) inspection (if available)
- Outputs:
  - A suspicion score ∈ [0, 1]

**Note**

This is **not a reliable AI detector**, only a heuristic indicator.

## Requirements

- C++17
- OpenCV ≥ 4.5 (recommended: 4.10+)
- CMake ≥ 3.10

Optional (for extended analysis):
- OpenCV contrib modules (for SIFT)

---

## Build Instructions

```bash
git clone <your-repo-url>
cd <project-folder>

mkdir build && cd build
cmake ..
make -j
```

---

## Usage

```bash
./image_analyzer ../data/in/image1.jpg ../data/in/image2.jpg
./image_analyzer ../data/in/image1.jpg ../data/in/image2.jpg ../data/out
```

## Methodology Summary

| Module                | Approach                          |
|---------------------|----------------------------------|
| Context Analysis     | Feature matching + homography    |
| Derivation Detection | Image alignment + differencing   |
| AI Suspicion         | Heuristic statistical analysis   |

## Disclaimer

This tool provides heuristic and probabilistic analysis only.

It must NOT be used for:
- legal decisions
- forensic certification
- proof of authenticity

Results should always be interpreted with caution.
